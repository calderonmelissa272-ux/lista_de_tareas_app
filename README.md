# Lista de Tareas con Tkinter

Aplicación desarrollada en Python utilizando Tkinter para gestionar tareas diarias.

## Funcionalidades
- Agregar tareas
- Marcar como completadas
- Eliminar tareas

## Atajos de teclado
- Enter: Agregar tarea
- C: Completar tarea
- D o Delete: Eliminar tarea
- Escape: Cerrar aplicación

## Arquitectura
El proyecto sigue una arquitectura modular por capas:
- modelos
- servicios
- interfaz gráfica (ui)
- main
